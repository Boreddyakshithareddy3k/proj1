## Overview

This project simplifies faculty registration processes while ensuring data integrity through innovative features such as PDF generation and email-linked password reset. It is a web application developed using PHP, HTML, CSS, and MySQL, combining backend robustness with a user-friendly frontend.

### Key Features

- **PDF Generation:** Users can generate PDFs using provided data, enhancing documentation and record-keeping.
  
- **Email-Linked Password Reset:** Implemented secure password reset functionality linked to user email addresses for enhanced account security and usability.

## Technologies Used
PHP, MySQL, HTML, CSS, Apache, XAMPP
